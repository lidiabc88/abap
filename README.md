✍️​ Portfolio ABAP – Desarrollo Backend SAP
Descripción

Este repositorio contiene ejercicios y desarrollos prácticos realizados durante mi formación en desarrollo backend SAP ABAP. El contenido está enfocado en conceptos modernos de ABAP utilizando Eclipse (ADT), incluyendo Open SQL, CDS Views y estructuras básicas de RAP.

El objetivo de este repositorio es mostrar conocimientos prácticos de desarrollo en SAP ABAP de forma estructurada y aplicada.

✨ Sobre mí

Certificado SAP Backend Developer – SAP Cloud Application Programming Model (CAP)
Certificado SAP Developer – ABAP (C_ABAPD / C_CPE)

Formación en desarrollo de aplicaciones web y más de 12 años de experiencia en entornos comerciales y trato con cliente. Actualmente enfocado en desarrollo backend SAP y programación de aplicaciones empresariales.
